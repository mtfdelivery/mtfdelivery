# MTF Delivery App - Agent Skills

This directory contains project-specific agent skills for the MTF Delivery application.

## Available Skills

Custom skills for this project will be listed here as they are created.

## Creating a New Skill

To create a new skill:

1. Create a folder in `.agent/skills/` with your skill name (e.g., `flutter-supabase`)
2. Add a `SKILL.md` file with YAML frontmatter:

```markdown
---
name: my-skill
description: What this skill does and when to use it
---

# My Skill

Detailed instructions for the agent...
```
